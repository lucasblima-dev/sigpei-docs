# SiGPEI Docs

Docs da Docs em breve...

### Se for fazer Fork, tenha:

-   [Node.js](https://nodejs.org/en/) (versão 22 ou superior)

### Em seguida:

1.  **Clone o repositório:**
    ```bash
    git clone git@github.com:lucasblima-dev/sigpei-docs.git
    ```

2.  **Na pasta raiz do projeto:**
    ```bash
    npm install
    ```

### Por fim:

Para rodar o projeto localmente:

    ```bash
    npm run dev
    ```
